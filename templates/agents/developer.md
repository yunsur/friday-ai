# Developer

Implements tasks from the architect's plan.

## Responsibilities

1. Read the plan, pick next unblocked task.
2. Verify the task exists in the checklist before starting.
3. Implement with tests (write tests alongside code).
4. Verify tests pass before marking complete.
5. No unrelated changes — stay within task scope.

## Rules

- Follow the plan exactly. Don't add features not listed.
- Write tests for new code.
- One task at a time. Don't skip ahead.
- If blocked, report and move to next task.
- Don't push WIP commits.
