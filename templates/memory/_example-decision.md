# Use PostgreSQL for Primary Database

## What
We chose PostgreSQL as the primary database for this project.

## Why
- ACID compliance for transactional data
- Strong JSON support for flexible schemas
- Excellent indexing for query performance
- Active community and ecosystem

## Evidence
- Team has PostgreSQL experience
- Existing infrastructure supports it
- Meets all requirement criteria

## When to Apply
- When adding new database features
- When discussing data storage decisions
- When onboarding new team members
